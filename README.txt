# 🚀 WinSysCare v1.1.0
**A Professional Windows Maintenance & Optimization Tool**

Developed by **Ahmed Rabea**

WinSysCare is a powerful, lightweight utility designed to keep Windows systems clean, fast, and stable. It combines deep cleaning, user profile management, startup optimization, and system repair tools into one modern interface.

---

## ✨ Key Features
- **📊 System Dashboard:** Real-time monitoring of CPU and RAM usage.
- **🧹 Deep Cleaner:** Removes junk files, Windows update cache, and system logs safely.
- **🚀 Startup Manager:** View applications that launch at Windows startup.
- **👥 User Profiles Manager:** Safely deletes inactive domain/local user profiles (Skips Administrator & Current User).
- **🛠️ System Repair:** Integrated SFC Scan to fix Windows files.

---

## 🛠️ Built With
- **Python 3.10+**
- **CustomTkinter** (Modern UI)
- **Psutil** (System Monitoring)
- **PowerShell Integration** (Safe Profile Deletion)
